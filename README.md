## 🎯 Overview

This platform enables organizations to manage their procurement
process from purchase requests to vendor payments.

### User Roles
| Role | Responsibilities |
|------|-----------------|
| **Employee** | Create and submit purchase requests |
| **Manager** | Review, approve or reject requests |
| **Vendor** | Submit quotations and upload invoices |
| **Admin** | Manage users, generate POs, process payments |
